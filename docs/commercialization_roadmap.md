# Commercialization roadmap

Phase 1: campus Power BI pilot. Phase 2: district template app. Phase 3: Power BI Embedded product. Phase 4: Dataverse/Power Apps commercial version. Phase 5: full SaaS web app with SIS integrations.

Future features: SIS API integration, College Board import automation, Cambium/STAAR import automation, TSIA import automation, automated transcript parser, AI explanation layer, student-facing pathway portal, parent-facing pathway summary, district benchmark alignment, intervention curriculum library, cohort forecasting, and accountability simulation.
