# Power BI build guide

1. Install/open Power BI Desktop.
2. Create parameter `pDataFolder`.
3. Load each CSV template or sample CSV.
4. Paste/recreate Power Query scripts.
5. Rename tables.
6. Set data types.
7. Build relationships.
8. Add calculated columns.
9. Add DAX measures.
10. Create report pages.
11. Add slicers and conditional formatting.
12. Add Power Apps visual for notes.
13. Publish to Power BI Service.
14. Configure row-level security.
15. Schedule refresh.
16. Validate calculations against source extracts and TEA/local accountability guidance.
