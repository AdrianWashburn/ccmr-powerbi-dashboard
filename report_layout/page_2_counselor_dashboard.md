# Page 2: Counselor Dashboard

Purpose: action-oriented CCMR monitoring and next-best-action decisions.

Audience: Counselor Dashboard users.

Slicers: Campus, cohort, grade, counselor, subgroup, pathway, action status, program of study.

Visuals: KPI cards, trend/stacked bars, student detail table, decomposition tree, and drillthrough to Student Profile. Include fields for CCMR status, primary pathway, backup pathway, next action, follow-up date, ELA/math gap, CTE level, IBC status, intervention status, attendance concern, and discipline concern as relevant to this page.

Conditional formatting: green met/verified, amber one step away/near cut, red no clear pathway/overdue/missing evidence, blue enrichment or extension.

Tooltips: best score, skill gap, evidence source, rule used, last note, action owner, and refresh timestamp.

Questions answered: current status, closest pathway, strengths, gaps, next course, next test, needed intervention, and ownership.
